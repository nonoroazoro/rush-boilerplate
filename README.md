<h1 align="center">rush-boilerplate</h1>
<p align="center">Just clone and rename to create your own Rush projects</p>

## How to use?

1. Install dependencies.

    ```bash
    pnpm install
    rush install
    ```

1. Start the project.

    ```bash
    npm start
    ```

1. Build.

    ```bash
    npm run build
    ```
